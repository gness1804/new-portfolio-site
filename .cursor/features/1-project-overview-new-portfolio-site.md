# Project Overview New Portfolio Site

## Working directory

`~/Desktop/new-portfolio-site`

## Contents

I want to create a new portfolio site. That is a personal site that advertises things like my work, resume, and personal writings and materials. It's been a long time since I've created such a site. I want to create a revamped site that displays the following: 
- My best projects with links to them and brief descriptions
- Some version of my resume. 
- An overview of my professional skill set and focus. 
- Probably some things related to AI, such as a discussion of what I've done with it. (Related to the Best Projects section.)
- A section for a personal blog. This blog could be about both technical and non-technical topics. For instance, I might want to post some of my pieces about religion and secularism on the space. 
- Room for other sections in the future.  

## Acceptance criteria

- A nice-looking portfolio site that has all of the information described above. 
- This will be a static site. As such, deployment should be less complicated than some other projects, like my AI projects. I might use something like AWS Amplify to deploy the site. Deployment should be relatively simple, but it should also be secure. 
- The site will be personally branded. My colors are black, gray/silver, and red. This should be the color scheme of the site. The site should also have a logo representing me. Right now, I'm thinking an outline of the head of a cat with those three main theme colors in it, in a striped format. 
- The site should have a custom URL that I can link people to in places such as my resume.
