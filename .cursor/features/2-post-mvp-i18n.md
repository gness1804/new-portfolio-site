# Post MVP: internationalization 

## Content

After completing the MVP of the project, I would like to explore the possibility of internationalization or localization. That is, translating the site in a language other than English. I'm thinking French and Spanish would be good choices. There would be a language dropdown to choose your language. When you choose a language, the site would reload in that language. If you're on a particular page and you choose another language, the same page will reload in the new language. The default language will be US English. However, we might look into automatically loading the language that matches the user's language setting in their browser, if available. If no language is set in the browser, or if English is set, or if a language other than English, Spanish, or French is set, we would just default to English.   

## Acceptance Criteria

- The site will be fully translated into French and Spanish, as well as English. 
- Users will be able to seamlessly switch between languages via a user-friendly toggle. 
- The site will default to English. BUT the site will automatically load in Spanish or French if the user's browser's primary language is one of those two. Otherwise, the site will load in English.
